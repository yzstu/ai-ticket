# 新增技术指标因子说明

## 📊 概述

本次更新在 `src/analysis/quick_analyzer.py` 中新增了4个重要的技术指标因子，显著提升了评分系统的准确性和全面性。

## 🎯 新增指标详情

### 1. ADX 趋势强度指标

**权重：10分**

**功能说明：**
- 测量趋势的强度而不考虑方向
- ADX > 25：趋势明确 (+10分)
- ADX > 20：趋势较明确 (+5分)
- ADX ≤ 20：趋势不明确 (0分)

**计算方法：**
```python
# 基于ATR（平均真实波幅）计算的简化ADX
volatility_pct = (atr / close_price) * 100
adx = min(50, volatility_pct * 10)
```

**实际应用：**
- 识别趋势性行情
- 区分震荡市和趋势市
- 提高趋势跟随策略的准确性

---

### 2. 相对强度 RS 指标

**权重：10分**

**功能说明：**
- 比较个股与大盘的表现
- RS > 1.1：明显跑赢大盘 (+10分)
- RS < 0.9：明显弱于大盘 (-5分)
- 1.1 ≥ RS ≥ 0.9：表现正常 (0分)

**计算方法：**
```python
# 10日相对强度 = 个股10日收益率 / 大盘10日收益率
stock_returns = (current_price / price_10d_ago - 1)
rs_10d = 1 + (stock_returns - market_returns) / market_returns
```

**实际应用：**
- 识别强势股票
- 判断个股在大盘中的相对表现
- 优选跑赢大盘的标的

---

### 3. 布林带位置指标

**权重：10分**

**功能说明：**
- 判断价格在布林带中的相对位置
- 位置 < 0.2：接近下轨，超卖 (+10分)
- 位置 > 0.8：接近上轨，超买 (-5分)
- 0.2 ≤ 位置 ≤ 0.8：正常区间 (0分)

**计算方法：**
```python
# 布林带位置 = (当前价格 - 下轨) / (上轨 - 下轨)
boll_position = (close - boll_lower) / (boll_upper - boll_lower)

# 布林带计算
ma20 = close.rolling(20).mean()
std20 = close.rolling(20).std()
boll_upper = ma20 + 2 * std20
boll_lower = ma20 - 2 * std20
```

**实际应用：**
- 发现超买超卖机会
- 寻找支撑阻力位
- 辅助买卖点判断

---

### 4. KDJ 金叉死叉指标

**权重：5分**

**功能说明：**
- 识别KDJ指标的金叉死叉信号
- K线上穿D线（金叉）：买入信号 (+5分)
- K线下穿D线（死叉）：卖出信号 (-5分)
- 其他情况：无信号 (0分)

**计算方法：**
```python
# RSV计算
rsv = (close - low_min) / (high_max - low_min) * 100

# K值和D值（简化计算）
kdj_k = rsv
kdj_d = (current_rsv + previous_rsv) / 2

# 金叉死叉判断
if kdj_k > kdj_d and prev_kdj_k <= prev_kdj_d:  # 金叉
    score += 5
elif kdj_k < kdj_d and prev_kdj_k >= prev_kdj_d:  # 死叉
    score -= 5
```

**实际应用：**
- 短期买卖信号识别
- 超买超卖辅助判断
- 提高交易时机把握

## 🔧 技术实现

### 数据处理增强

在 `fast_analyze_stocks()` 函数中新增了：

1. **ADX计算模块**
   - 基于ATR的真实波动率计算
   - 转换为0-50范围的ADX值
   - 14日周期计算

2. **RS计算模块**
   - 10日个股与大盘收益率对比
   - 相对强度量化
   - 动态基准比较

3. **布林带计算模块**
   - 20日移动平均线
   - 2倍标准差上下轨
   - 价格位置百分比

4. **KDJ计算模块**
   - 9日周期RSV
   - K值D值平滑计算
   - 金叉死叉信号识别

### 容错机制

所有新指标都包含完善的容错处理：

```python
# 示例：ADX容错
if len(daily_data) >= 14:
    # 计算ADX
    stock_data['adx'] = calculated_adx
else:
    # 数据不足时使用默认值
    stock_data['adx'] = 20  # 默认中等趋势强度
```
