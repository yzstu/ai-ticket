# 回测可视化功能 - 快速启动指南

## 🚀 启动步骤

### 1. 启动服务
```bash
cd /Users/baldwin/PycharmProjects/ai-ticket
python main.py
```

服务启动后显示：
```
INFO: Started server process [12345]
INFO: Waiting for application startup.
INFO: Application startup complete.
INFO: Uvicorn running on http://0.0.0.0:8000
```

### 2. 访问前端
打开浏览器访问：**http://localhost:8000**

### 3. 验证功能

#### 3.1 生成推荐记录
1. 点击顶部导航"AI分析"标签页
2. 选择分析模式（如"Top-N 快速分析"）
3. 设置最大结果数（如20）
4. 点击"开始分析"按钮
5. 等待分析完成（可以看到"已自动保存 X 条推荐记录"的日志）

#### 3.2 查看回测可视化
1. 点击顶部导航"回测分析"标签页
2. 查看四个可视化区域：

**回测成功率总览**
- 显示总体成功率、总推荐数等关键指标

**成功率趋势**
- 折线图显示成功率随时间变化
- 柱状图显示每日推荐数量
- 可切换时间间隔：按日/按周/按月
- 可切换统计天数：7天/30天/90天/180天

**评分分布统计**
- 水平条形图显示6个评分区间的成功率
- 绿色（高成功率）到红色（低成功率）
- 显示每个区间的推荐数量

**成功率热力图**
- 表格显示周一到周五的成功率
- 单元格颜色：绿色（≥70%）→ 黄色（50-70%）→ 红色（<50%）

## 📊 数据说明

### 推荐记录自动保存
- 每次执行股票分析后，推荐记录自动保存到数据库

- 日志中显示保存数量

### 回测数据生成
1. 先执行股票分析生成推荐记录
2. 执行回测计算收益率
3. 查看可视化图表

## 🔧 常见问题

### Q: 图表显示"暂无数据"
**A:** 需要先执行股票分析和回测，生成数据后即可显示

### Q: API返回404错误
**A:** 请确认服务已启动（http://localhost:8000 可访问）

### Q: 图表样式异常
**A:** 强制刷新浏览器（Ctrl+F5 或 Cmd+Shift+R）清除缓存

### Q: 如何查看数据库记录？
```bash
# 查看推荐记录
sqlite3 ./cache/backtest.db "SELECT * FROM recommendations ORDER BY created_at DESC LIMIT 10;"

# 查看回测结果
sqlite3 ./cache/backtest.db "SELECT * FROM backtest_results ORDER BY created_at DESC LIMIT 10;"
```
