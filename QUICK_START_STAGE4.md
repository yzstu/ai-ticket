## 📊 功能说明

### 动态权重工作原理

**1. 市场环境检测**
- 实时获取上证指数数据
- 基于MA5/MA20/MA60均线判断
- 自动识别：牛市/熊市/震荡市

**2. 权重应用**
```
牛市: 技术50% + 情绪30% + 资金20%
熊市: 技术70% + 情绪10% + 资金20%
震荡: 技术40% + 情绪20% + 资金40%
```

**3. 智能建议**
- 不同环境下使用不同阈值
- 牛市更积极，熊市更保守
