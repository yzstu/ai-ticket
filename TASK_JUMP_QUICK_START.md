## 🎯 快速体验步骤

### 1️⃣ 打开系统
访问 http://localhost:8000/web

### 2️⃣ 进入任务中心
点击顶部导航的 **"任务中心"** 标签页

### 3️⃣ 创建任务
点击 **"快速创建"** 区域的任意任务按钮：
- 📊 股票分析
- 📈 批量回测
- 💾 缓存预加载
- 🏢 板块分析

### 4️⃣ 等待完成
观察任务状态从 **"RUNNING"** 变为 **"COMPLETED"**

### 5️⃣ 查看详情
点击任务卡片的 **"详情"** 按钮

### 6️⃣ 使用跳转
在任务详情对话框中：
- 查看 **"任务结果"** 区域的跳转链接
- 查看 **"结果摘要"** 区域的关键数据
- 点击跳转链接或 **"查看完整结果"** 按钮

### 7️⃣ 验证跳转
系统会自动：
- 关闭任务详情对话框
- 切换到对应的功能页面
- 自动设置页面参数（如需要）
- 显示通知消息

## 📋 支持的任务类型

| 任务类型 | 跳转目标 | 传递参数 |
|---------|---------|---------|
| 📊 股票分析 | AI智能分析页面 | selection_mode, max_results |
| 📈 批量回测 | 回测分析页面 | 无 |
| 💾 缓存预加载 | 系统缓存页面 | 无 |
| 🏢 板块分析 | 热门板块页面 | sector_name |

## ⚠️ 注意事项

### 显示条件
- **仅已完成任务** 显示跳转链接
- 未完成或失败的任务不会显示

### 跳转行为
- 点击跳转会**自动关闭**任务详情对话框
- 部分跳转会**自动设置**页面参数
- 跳转后会显示**确认通知**

### 浏览器要求
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 🎓 使用技巧

### 1. 快速创建分析任务
1. 在任务中心点击 **"分析"** 按钮
2. 设置参数：Top-N / 自定义 / 范围 / 全量
3. 点击创建
4. 等待完成
5. 查看详情并跳转

### 2. 对比不同模式结果
1. 创建多个不同模式的分析任务
2. 等待全部完成
3. 逐个查看详情
4. 点击跳转到分析页面对比结果

### 3. 查看缓存效果
1. 创建缓存预加载任务
2. 等待完成
3. 点击跳转到缓存页面
4. 查看缓存统计和效果

### 4. 板块投资分析
1. 在热门板块页面选择板块
2. 或直接创建板块分析任务
3. 等待完成
4. 点击跳转到板块页面
5. 查看板块内股票表现

## 📞 技术支持

### 常见问题

**Q: 为什么看不到跳转链接？**
A: 请确认任务状态为 COMPLETED（已完成）

**Q: 跳转后页面参数没有自动设置？**
A: 请检查目标页面是否已加载完成，等待 500ms

**Q: 动画效果不流畅？**
A: 请升级浏览器到最新版本

**Q: 通知消息不显示？**
A: 请检查浏览器是否允许通知

- 系统日志: `server.log`
